В новой версии в объявлении класса находятся только прототипы функций,
а сами функции вынесены из объявления. Это сильно повышает читаемость кода.
И вообще это соответсвует стилю Си++

Добавлена функция нормировки. Теперь картину всегда будет хорошо видно, 
потому что интенсивность всегда будет лежать в диапозоне от 0 (самая тусклая точка из вычисляемых) 
до 255 (самая яркая точка из вычисляемых).

Функция подчета времени незначительно поменялась. Теперь нет (int) потому что оно и не требовалось
